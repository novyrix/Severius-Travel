# Severius Adventures & Travel Logo

## Required Logo File

Please place your Severius phoenix logo file here as `logo.png`.

### Logo Requirements:
- **Filename**: `logo.png`
- **Format**: PNG with transparent background
- **Recommended Size**: 600x600px (minimum 400x400px)
- **Design**: Phoenix bird with golden wings on brown background or transparent
- **Color Scheme**: 
  - Primary Brown: #4e342e (rgb(78, 52, 46))
  - Primary Gold: #d4af37 (rgb(212, 175, 55))

### Usage
This logo is used in:
- All email templates (200px width in emails)
- Site favicon (needs conversion to favicon.ico)
- Site metadata (og:image)
